# Lector
Qt based ebook reader

Currently supports:
* epub
* mobi
* azw / azw3 / azw4
* cbr / cbz

## Requirements
* Qt5
* PyQt5 
* python-requests
* python-beautifulsoup4

## Installation
0. Install dependencies
1. Clone repository
2. Launch with \_\_main\_\_.py
